**GOG CLI basics:**
- `gog auth credentials /path/to/client_secret.json` - Load OAuth credentials
- `gog auth add email --services gmail,calendar,drive,contacts,sheets,docs` - Add account
- `gog gmail search 'newer_than:1d'` - Search emails
- `gog gmail get <messageId>` - Read specific email
- `gog calendar create primary --summary "Event" --from 2026-01-23T11:00:00+08:00 --to 2026-01-23T12:00:00+08:00` - Create event

**Authentication notes:**
- Requires keyring passphrase for secure token storage
- Set `GOG_KEYRING_PASSWORD` environment variable for non-interactive use
- Test users must be added in Google Cloud Console for development
